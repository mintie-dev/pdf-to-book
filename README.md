# PDF to Book

Transform your PDF files into an immersive, book-like reading experience.

## Features

- 📚 **Personal Digital Library** – Upload PDFs and organize them as a beautiful book collection
- 🔍 **Book Discovery** – Search and add books from Open Library
- 📖 **Immersive Reader** – Read PDFs in a clean, distraction-free interface
- 📊 **Reading Progress** – Real-time tracking as you scroll through pages
- 🔥 **Streaks & Goals** – Set daily page goals and track your reading streak (inspired by Duolingo)
- 📝 **Reading Log** – GitHub-style contribution graph showing your reading activity
- 🔖 **Bookmarks & Highlights** – Save your place and highlight important passages
- 📕 **Want to Read** – Save books you discover for later
- 🎨 **Themes** – Light, Dark, and Warm Blush reading modes
- 📱 **Drag & Drop** – Simply drag PDFs onto the app to start reading

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and builds
- **Tailwind CSS** for styling
- **shadcn/ui** for UI components
- **pdf.js** for PDF rendering
- **Open Library API** for book search and discovery

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
